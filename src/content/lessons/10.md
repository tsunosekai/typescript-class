## 配列とは？

**配列**とは、「複数のデータを順番に入れておける箱」です。

例えば、3人の名前を管理したい場合：

```typescript
// 変数を3つ使う方法（面倒…）
const name1 = "山田";
const name2 = "佐藤";
const name3 = "田中";

// 配列を使う方法（スッキリ！）
const names = ["山田", "佐藤", "田中"];
```

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];
```

- `string[]` → 「string型の配列」という意味
- `number[]` → 「number型の配列」

```typescript
const numbers: number[] = [1, 2, 3, 4, 5];

```

## STEP 4: 要素にアクセスする

配列の中の各データを「**要素**」と呼びます。

要素には「**インデックス**（番号）」でアクセスします。

### インデックスは0から始まる！

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];
//                         [0]       [1]        [2]

console.log(fruits[0]);  // りんご
console.log(fruits[1]);  // バナナ
console.log(fruits[2]);  // オレンジ
```

> **注意！**
> プログラミングでは、番号は **0から始まります**。
> 1番目 → インデックス0
> 2番目 → インデックス1
> 3番目 → インデックス2

### 要素を変更する

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];
fruits[1] = "ぶどう";  // バナナ → ぶどう

console.log(fruits);  // [ 'りんご', 'ぶどう', 'オレンジ' ]
```

## STEP 5: 配列の長さを知る

`length` プロパティで、配列の要素数がわかります：

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];
console.log(fruits.length);  // 3
```

## STEP 6: 要素を追加・削除する

### 末尾に追加: push

```typescript
const fruits: string[] = ["りんご", "バナナ"];
fruits.push("オレンジ");

console.log(fruits);  // [ 'りんご', 'バナナ', 'オレンジ' ]
```

### 末尾から削除: pop

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];
const removed = fruits.pop();

console.log(removed);  // オレンジ（削除された要素）
console.log(fruits);   // [ 'りんご', 'バナナ' ]
```

### 先頭に追加: unshift

```typescript
const fruits: string[] = ["バナナ", "オレンジ"];
fruits.unshift("りんご");

console.log(fruits);  // [ 'りんご', 'バナナ', 'オレンジ' ]
```

### 先頭から削除: shift

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];
const removed = fruits.shift();

console.log(removed);  // りんご
console.log(fruits);   // [ 'バナナ', 'オレンジ' ]
```

## STEP 7: 配列をループで処理する

配列の全要素に対して処理を行いたい時は、**ループ**を使います。

### for文

> **`for` ループとは？**
> 「○回繰り返す」という処理には `for` ループを使います。
> ```typescript
> for (let i = 0; i < 3; i++) {
>     // 3回繰り返す処理
> }
> ```
> - `let i = 0` → カウンター変数 `i` を0から始める
> - `i < 3` → `i` が3より小さい間、繰り返す
> - `i++` → 毎回 `i` を1増やす（`i = i + 1` と同じ意味）

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];

for (let i = 0; i < fruits.length; i++) {
    console.log(`${i + 1}番目: ${fruits[i]}`);
}
```

結果：
```
1番目: りんご
2番目: バナナ
3番目: オレンジ
```

### for...of文（おすすめ！）

インデックスが不要な場合は、`for...of` が簡単です：

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];

for (const fruit of fruits) {
    console.log(fruit);
}
```

結果：
```
りんご
バナナ
オレンジ
```

### forEachメソッド

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];

fruits.forEach((fruit, index) => {
    console.log(`${index + 1}番目: ${fruit}`);
});
```

> **forEachの書き方を分解すると…**
> ```typescript
> fruits.forEach((fruit, index) => {
>     // ここに処理を書く
> });
> ```
> - `fruits.forEach(...)` → 配列の各要素に対して処理を実行
> - `(fruit, index) => { ... }` → 第9回で学んだアロー関数
> - `fruit` → 配列の各要素が順番に入る
> - `index` → 何番目か（0から始まる）が入る
>
> **for...of の方がシンプルなので、最初はそちらを使ってOK！**

## 便利な配列メソッド

> **メソッドとは？**
> 配列やオブジェクトに「くっついている関数」のことです。
> `配列.メソッド名()` の形で使います。さっき使った `push` や `pop` もメソッドです。

### includes: 含まれているか

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];

console.log(fruits.includes("バナナ"));  // true
console.log(fruits.includes("ぶどう"));  // false
```

### indexOf: 位置を調べる

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];

console.log(fruits.indexOf("バナナ"));  // 1
console.log(fruits.indexOf("ぶどう"));  // -1（見つからない）
```

### join: 文字列に結合

```typescript
const fruits: string[] = ["りんご", "バナナ", "オレンジ"];

console.log(fruits.join(", "));  // りんご, バナナ, オレンジ
```
