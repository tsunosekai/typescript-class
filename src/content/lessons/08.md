## 条件分岐とは？

条件分岐とは、「**条件によって処理を変える**」ことです。

例えば：
- もし18歳以上なら → 「大人です」と表示
- もし18歳未満なら → 「子どもです」と表示

こういった処理を書けるのが **if文** です。

---

## STEP 3: 比較演算子を覚えよう

条件を書くには「**比較演算子**」を使います。

| 演算子 | 意味 | 例 |
|--------|------|-----|
| `===` | 等しい | `age === 18` |
| `!==` | 等しくない | `age !== 18` |
| `<` | より小さい | `age < 18` |
| `>` | より大きい | `age > 18` |
| `<=` | 以下 | `age <= 18` |
| `>=` | 以上 | `age >= 18` |

試してみましょう：

```typescript
const age: number = 16;

console.log(age === 16);  // true（等しい）
console.log(age === 18);  // false（等しくない）
console.log(age < 18);    // true（16は18より小さい）
console.log(age >= 16);   // true（16は16以上）
```

**`Ctrl + S`** で保存して実行：

```
node condition.ts
```

## STEP 4: if文の基本

### 4-1. 基本の形

```typescript
if (条件) {
    // 条件がtrueの時に実行される
}
```

### 4-2. 実際に書いてみよう

`condition.ts` を書き換えます：

```typescript
const age: number = 16;

if (age < 18) {
    console.log("未成年です");
}
```

**`Ctrl + S`** で保存して実行：

```
node condition.ts
```

結果：
```
未成年です
```

`age` が18より小さいので、中のコードが実行されました。

## STEP 5: elseを使う

`else` を使うと、条件がfalseの時の処理を書けます。

```typescript
const age: number = 20;

if (age < 18) {
    console.log("未成年です");
} else {
    console.log("成人です");
}
```

**`Ctrl + S`** で保存して実行：

```
成人です
```

`age` が20なので、`else` の中が実行されました。

## STEP 6: else ifを使う

条件が複数ある場合は `else if` を使います。

```typescript
const score: number = 75;

if (score >= 90) {
    console.log("素晴らしい！A評価です");
} else if (score >= 70) {
    console.log("よくできました！B評価です");
} else if (score >= 50) {
    console.log("がんばりましょう。C評価です");
} else {
    console.log("もう少し頑張ろう。D評価です");
}
```

**`Ctrl + S`** で保存して実行：

```
よくできました！B評価です
```

`score` が75なので、`score >= 70` の条件に当てはまりました。

## STEP 7: 論理演算子を使う

複数の条件を組み合わせたい時は「**論理演算子**」を使います。

### AND（かつ）: `&&`

**両方の条件がtrue**の時にtrueになります。

```typescript
const age: number = 16;
const hasPermission: boolean = true;

if (age >= 13 && hasPermission) {
    console.log("このゲームをプレイできます");
}
```

### OR（または）: `||`

**どちらかの条件がtrue**の時にtrueになります。

```typescript
const isStudent: boolean = true;
const isSenior: boolean = false;

if (isStudent || isSenior) {
    console.log("割引が適用されます");
}
```

### NOT（否定）: `!`

条件を**反転**させます。

```typescript
const isLoggedIn: boolean = false;

if (!isLoggedIn) {
    console.log("ログインしてください");
}
```

## Math.random() について

> **`Math` って何？**
> `Math` は、数学的な計算をするための便利な道具箱です。
> - `Math.random()` → ランダムな数を作る
> - `Math.floor()` → 小数点以下を切り捨てる
> - `Math.round()` → 四捨五入する
>
> 最初から使えるように用意されているので、インストールなどは不要です。

```typescript
Math.random()
```
→ 0以上1未満のランダムな小数を返します（例: 0.7392...）

```typescript
Math.random() * 100
```
→ 0以上100未満のランダムな小数（例: 73.92...）

```typescript
Math.floor(Math.random() * 100)
```
→ 小数点以下を切り捨てて整数に（例: 73）

## よくある間違い

### `=` と `===` を間違える

```typescript
// ❌ 間違い（代入になってしまう）
if (age = 18) { }

// ✅ 正しい（比較）
if (age === 18) { }
```

### `==` ではなく `===` を使う

```typescript
// ⚠️ 使わない方がいい（型変換が起きる）
if (age == "18") { }

// ✅ こちらを使う（厳密な比較）
if (age === 18) { }
```

TypeScriptでは `===`（厳密等価演算子）を使いましょう。
