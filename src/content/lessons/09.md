---

## なぜ繰り返し処理が必要？

たとえば「こんにちは」を5回表示したい場合：

```typescript
// 繰り返しを使わない方法（大変…）
console.log("こんにちは");
console.log("こんにちは");
console.log("こんにちは");
console.log("こんにちは");
console.log("こんにちは");
```

これが100回だったら？ 1000回だったら？

**for文**を使えば：

```typescript
// 繰り返しを使う方法（スッキリ！）
for (let i = 0; i < 5; i++) {
    console.log("こんにちは");
}
```

たった3行で済みます！

## STEP 3: for文の基本を学ぼう

### 3-1. for文の形

```typescript
for (let i = 0; i < 5; i++) {
    // 繰り返したい処理
}
```

これを分解すると：

| 部分 | 意味 |
|------|------|
| `let i = 0` | カウンター変数 `i` を0から始める |
| `i < 5` | `i` が5より小さい間、繰り返す |
| `i++` | 毎回 `i` を1増やす |

### 3-2. 実際に動かしてみよう

`loops.ts` に以下を入力：

```typescript
for (let i = 0; i < 5; i++) {
    console.log("こんにちは");
}
```

**`Ctrl + S`** で保存して実行：

```
node loops.ts
```

結果：
```
こんにちは
こんにちは
こんにちは
こんにちは
こんにちは
```

5回表示されました！

<details>
<summary>🔧 for文がうまく動かないときは</summary>

**エラー例①：無限ループになる（プログラムが止まらない）**
```typescript
// ❌ これは無限ループ！
for (let i = 0; i < 5; ) {  // i++ がない！
    console.log(i);
}
```
→ `i++` を忘れていないか確認してください
→ 無限ループになったら `Ctrl + C` で強制終了できます

**エラー例②：一度も実行されない**
→ 開始値と条件を確認してください（`i = 10` から始まり `i < 5` だと最初から false）

**エラー例③：想定より1回多い/少ない**
→ `<` と `<=` の違いに注意！
  - `i < 5` → 0〜4 の5回
  - `i <= 5` → 0〜5 の6回

**よくある間違い**
- セミコロンとカンマの混同: `for (let i = 0, i < 5, i++)` → `for (let i = 0; i < 5; i++)`
- 波括弧 `}` の閉じ忘れ

</details>

## STEP 4: カウンター変数を使う

カウンター変数 `i` は、ループの中で使えます。

```typescript
for (let i = 0; i < 5; i++) {
    console.log(`${i}回目のループ`);
}
```

結果：
```
0回目のループ
1回目のループ
2回目のループ
3回目のループ
4回目のループ
```

> **注意！**
> `i` は **0から始まる** ので、0〜4の5回繰り返されます。
> プログラミングでは、番号は0から始めることが多いです。

### 1から始めたい場合

```typescript
for (let i = 1; i <= 5; i++) {
    console.log(`${i}回目のループ`);
}
```

結果：
```
1回目のループ
2回目のループ
3回目のループ
4回目のループ
5回目のループ
```

`i = 1` で始めて、`i <= 5`（5以下）で終わるようにします。

## STEP 5: いろいろな繰り返しを試そう

### 2ずつ増やす

```typescript
for (let i = 0; i < 10; i += 2) {
    console.log(i);
}
```

結果：
```
0
2
4
6
8
```

> **`i += 2` とは？**
> `i = i + 2` と同じ意味です。「`i` に2を足して、`i` に代入する」という意味。

### 逆順（カウントダウン）

```typescript
for (let i = 5; i > 0; i--) {
    console.log(i);
}
console.log("発射！");
```

結果：
```
5
4
3
2
1
発射！
```

> **`i--` とは？**
> `i = i - 1` と同じ意味です。「`i` から1を引いて、`i` に代入する」という意味。

## STEP 7: for文とif文を組み合わせる

繰り返しの中で条件分岐もできます。

```typescript
// 1から10までの数字で、偶数だけ表示
for (let i = 1; i <= 10; i++) {
    if (i % 2 === 0) {
        console.log(`${i}は偶数です`);
    }
}
```

結果：
```
2は偶数です
4は偶数です
6は偶数です
8は偶数です
10は偶数です
```

> **`%` とは？**
> 割り算の「余り」を求める演算子です。
> - `10 % 3` → 1（10÷3=3 余り1）
> - `8 % 2` → 0（8÷2=4 余り0）
>
> 「2で割って余りが0」なら偶数です。

## STEP 8: 合計を計算する

繰り返しで合計を求めることもよくあります。

```typescript
// 1から10までの合計を計算
let total = 0;

for (let i = 1; i <= 10; i++) {
    total = total + i;  // total += i; と書いてもOK
}

console.log(`1から10までの合計: ${total}`);
```

結果：
```
1から10までの合計: 55
```

## 【補足】while文

`for`文以外に、`while`文という繰り返しもあります。

```typescript
let count = 0;

while (count < 5) {
    console.log(count);
    count++;
}
```

結果：
```
0
1
2
3
4
```

- `for`文：繰り返す回数が決まっているとき
- `while`文：条件が満たされている間ずっと繰り返すとき

最初は `for`文だけ覚えればOK！
